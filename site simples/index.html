Trocador de Tema

Sobre Esta Página

Esta página demonstra o uso de temas claro e escuro utilizando variáveis CSS e JavaScript. Você pode clicar no botão acima para alternar entre os dois temas.

Cada tema altera a cor de fundo, a cor do texto e os estilos do botão para proporcionar uma experiência visual diferente. O tema claro usa um fundo verde, enquanto o tema escuro usa um fundo preto.

Recursos do Trocador de Temas

  • Troca dinâmica de temas com um único clique.
  • Variáveis CSS para gerenciar cores e estilos.
  • Lógica em JavaScript para alternar entre temas.
